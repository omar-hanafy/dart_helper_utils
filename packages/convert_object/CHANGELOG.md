# Changelog

## 1.0.0

- Initial release
- Core `Converter` class with fluent API
- Support for primitive type conversions (String, int, double, num, bool)
- BigInt conversion support
- DateTime and Uri conversions
- Collection conversions (List, Set, Map)
- Rich extension methods for Iterables and Maps
- `ConversionResult` for functional error handling
- Custom converter support
- Comprehensive null-safety with OrNull and Or variants
- Nested data extraction capabilities
- JSON string decoding support
- Let extensions for null-safe transformations
- Type checking with `canConvertTo`
- Alternative keys/indices support for fault-tolerant parsing
